# Plegamiento-Proteico
Este proyecto simula la organización espacial de aminoácidos en una proteína utilizando Python.

## Descripción
Los aminoácidos se clasifican en:
- Hidrofóbicos (H)
- Polares (P)
- Cargados positivos (+)
- Cargados negativos (-)

Cada grupo se distribuye en círculos concéntricos para simular el plegamiento.

## Visualización
Se utiliza NetworkX y Matplotlib para representar los aminoácidos como un grafo, donde:
- Cada nodo es un aminoácido
- Las conexiones representan la secuencia
- Los colores representan el tipo
